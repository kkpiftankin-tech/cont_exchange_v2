# F-11 — External Venues LOB → FOB

> **Статус:** Not implemented (есть симулятор для smoke-тестов).

## Описание

Подключение к внешним торговым площадкам (Binance, Coinbase, ...), преобразование их LOB в FOB-представление и подача в matching solver как источник внешней ликвидности.

## Acceptance Criteria (IN-001)

Система должна получать и использовать рыночные данные от внешних бирж и пулов (CEX/DEX/AMM): LOB-снапшоты, цены, depth, комиссии.

- LOB → FOB конвертация даёт эквивалентные CSLO-кривые.
- Данные нормализованы и опубликованы в `marketdata.raw`.
- Используются как reference price для clearing и как опциональный источник ликвидности.
